# Ekam Bitt Portfolio

A premium, interactive developer portfolio built with the latest web technologies. Features a custom 3D Ethereum logo, vertex-based geometry, and a clean, responsive design.

## Tech Stack

-   **Framework**: Next.js 16 (App Router)
-   **Language**: TypeScript
-   **Styling**: Tailwind CSS v4
-   **3D Graphics**: Three.js, React Three Fiber, React Three Drei
-   **Animations**: Framer Motion
-   **Fonts**: Geist Sans, Geist Mono, Space Grotesk
-   **Linting**: ESLint 8 (with `eslint-config-next`)

## Getting Started

1.  **Install dependencies:**

    ```bash
    npm install
    ```

2.  **Run the development server:**

    ```bash
    npm run dev
    ```

    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

3.  **Linting:**

    To check for code quality issues:

    ```bash
    npx eslint .
    ```

4.  **Build for production:**

    ```bash
    npm run build
    ```

## Features

-   **Interactive 3D Hero**: A vertex-accurate 3D Ethereum logo that responds to mouse movement.
-   **Project Showcase**: 3D cards with hover effects and dynamic content.
-   **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
-   **Dark Mode**: Built-in dark mode support.
-   **SEO Optimized**: Includes dynamic sitemap, robots.txt, and comprehensive metadata.

## Deployment

This project is optimized for deployment on [Vercel](https://vercel.com).

1.  Push your code to a Git repository.
2.  Import the project into Vercel.
3.  Vercel will automatically detect Next.js and deploy.
